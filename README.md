# 2048

Una implementación sencilla del clásico juego 2048, hecha con HTML, CSS y JavaScript puro (sin dependencias).

## Cómo jugar

Abre `index.html` en tu navegador. Junta fichas iguales para llegar a la ficha **2048**.

- **Escritorio:** usa las flechas del teclado (↑ ↓ ← →).
- **Móvil:** desliza el dedo sobre el tablero en la dirección deseada.

Pulsa **Nuevo juego** en cualquier momento para reiniciar.

## Archivos

- `index.html` — estructura de la página.
- `style.css` — estilos del tablero y las fichas.
- `script.js` — lógica del juego (movimientos, combinaciones, puntuación).

La puntuación más alta se guarda en el `localStorage` del navegador.
